# SFP Study 1.5

Prototipo web installabile e utilizzabile anche offline dopo il primo caricamento.

## Funzioni
- Dashboard personale
- Gestione materie
- Caricamento e catalogazione locale dei materiali
- Creazione guidata delle lezioni
- Sincronizzazione automatica tra lezioni, manuale e registro
- Piano esami
- Esportazione del manuale in formato testo
- Tema chiaro/scuro e animazione con stelle lilla
- Salvataggio locale nel browser

## Avvio
Aprire `index.html` con un browser moderno. Per un uso più stabile, pubblicare la cartella con GitHub Pages o avviare un server locale.
